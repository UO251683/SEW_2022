Soporte de BBDD

Node.js admite todo tipo de bases de datos, sin importar si se trata de una base de datos relacional o una base de datos NoSQL. Sin embargo, las bases de datos NoSQL como MongoDb son las que mejor se adaptan a Node.js. Para acceder a la base de datos desde Node.js, primero debe instalar los controladores para la base de datos que desea usar. Las siguientes listas muestran las bases de datos relacionales y NoSQL más importantes y con mejor soporte para Node.JS:

Bases de Datos relacionales:

  • MS SQL Server
  • Oracle
  • MySQL
  • PostgreSQL
  • SQLite

Bases de Datos NoSQL:

  • MongoDB
  • Cassandra
  • LevelDB
  • RavenDB
  • Neo4j
  • Redis
  • CouchDB
